# 🚀 Java OOPs Concepts Implementation

Welcome to the Java OOPs repository! Yeh repository un sabhi core Object-Oriented Programming concepts ko cover karti hai jo Java seekhne ke liye sabse zaroori hain. 

Har ek `.java` file ek specific concept ko simple tarike se explain karne ke liye likhi gayi hai.

## 📚 Topics Covered in this Repository:

*   **Classes and Objects:** `ClassObject.java`, `Student.java`
*   **Encapsulation:** Data hiding aur getters/setters ka use (`Encapsulation.java`).
*   **Inheritance:** 
    *   Single Inheritance (`SingleInheritance.java`)
    *   Multilevel Inheritance (`MultilevelInheritance.java`)
    *   Hierarchical Inheritance (`HierarchicalInheritance.java`)
    *   Multiple Inheritance using Interfaces (`MultipleInheritance.java`)
*   **Polymorphism:**
    *   Compile-time / Method Overloading (`Compile_Time_Polymorphism.java`, `Method_Overloading.java`)
    *   Run-time / Method Overriding (`Runtime_Polymorphism.java`, `Method_Overriding.java`)
*   **Abstraction:** Abstract classes aur Interfaces ka implementation (`Abstract.java`, `Interface.java`).
